## Challenge Rick and Morty - React Native

### ¿Cómo se usa la app?

Para comenzar toque el boton ENTER, en el buscador escriba el nombre de algún personaje, lugar o episodio, luego puede variar entre ellos con los filtros ubicados en la parte inferior. Para borrar toda la búsqueda actual toque el icono rojo de la derecha del buscador. En caso de no haber resultados para su búsqueda se mostrará el mensaje "No results❌" Enjoy it! 🚀
